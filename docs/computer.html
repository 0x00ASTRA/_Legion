Computer.py Documentation

Overview

The `computer.py` file defines a Computer class and its components, including CPU, GPU, RAM, and Storage. Each of these is a class with its own attributes.

Classes

Computer Class

The Computer class represents a computer. It has attributes for CPU, GPU, RAM, storage, and whether it is compromised.

CPU Class

The CPU class represents a CPU. It has attributes for brand, model, and speed.

GPU Class

The GPU class represents a GPU. It has attributes for brand, model, and memory.

RAM Class

The RAM class represents RAM. It has an attribute for size.

Storage Class

The Storage class represents storage. It has attributes for type and size.
