CryptoCurrency.py Documentation

Overview

The `crypto_currency.py` file defines a CryptoCurrency class, which represents a cryptocurrency. The price of the cryptocurrency fluctuates randomly to simulate demand and supply.

Classes

CryptoCurrency Class

The CryptoCurrency class represents a cryptocurrency. It has attributes for name, symbol, and price. It also has a `update_price` method to update the price of the cryptocurrency and a `to_json` method to convert the cryptocurrency to a JSON-serializable format.
