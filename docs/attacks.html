Attacks.py Documentation

Overview

The `attacks.py` file defines various types of cyber attacks, including Virus, Ransomware, Worm, Phishing, and Exploit. Each of these is a class that inherits from the base `Attack` class.

Classes

Attack Class

The Attack class is the base class for all types of cyber attacks. It has a `name` attribute and a `__init__` method to set the name.

Virus Class

The Virus class represents a virus attack. It inherits from the Attack class and sets the name to "Virus".
