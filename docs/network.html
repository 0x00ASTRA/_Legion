Network.py Documentation

Overview

The `network.py` file defines a Node class and a TorNetwork class. The Node class represents a node in the Tor network, and the TorNetwork class represents the Tor network itself.

Classes

Node Class

The Node class represents a node in the Tor network. It has attributes for id, online status, compromised status, and reputation. It also has methods to go offline, go online, and get compromised.

TorNetwork Class

The TorNetwork class represents the Tor network. It has an attribute for nodes. It also has methods to add a node, remove a node, and send a packet.
